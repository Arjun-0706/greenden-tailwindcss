# 🌿 Greenden - Plant Store Website

Greenden is a simple **frontend plant store website** built using **Tailwind CSS** and **CSS Grid**.
The project focuses on creating a clean UI layout for an online plant shop with responsive design.

---

## 🚀 Live Demo

🔗 https://arjun-0706.github.io/greenden-tailwindcss/

---

## ✨ Features

* 🌱 Modern plant store UI
* 📱 Responsive layout
* 🎨 Styled using Tailwind CSS
* 🧩 Layout built using CSS Grid
* 🖼 Product cards with hover effects
* 🧭 Sticky navigation bar

---

## 🛠 Technologies Used

* HTML5
* Tailwind CSS
* CSS Grid

## 💻 How to Run

1. Clone the repository

```
git clone https://github.com/Arjun-0706/greenden-tailwindcss.git
```

2. Open the project folder

3. Run `index.html` in your browser
   (or use VS Code Live Server)
